O projeto é composto pelos seguintes elementos:

- "1_gerar_despesas", a main do programa 1
- "2_analisar_despesas", a main do programa 2
- "Funcoes.c", o ficheiro onde são criadas todas as funções necessárias para os dois programas correrem;
- "Estruturas.h", onde são criadas as estruturas e listas;
- "Declaracoes.h", onde são declaradas as funções;
- "Despesas.data" e "Orcamentos.data", os dois ficheiros binários de entrada;
- "config.txt", o ficheiro de saída;
- "despesas_totais" e "desvio_orcamento", os dois ficheiros pertencentes ao "config.txt";


Para o programa 1 executar, ele necessita de compilar em conjunto:
- "1_gerar_despesas.c", "Funcoes.c", "Declaracoes.h", "Estruturas.h".

Para o programa 2 executar, ele necessita de compilar em conjunto:
- "2_analisar_despesas.c", "Funcoes.c", "Declaracoes.h", "Estruturas.h".
